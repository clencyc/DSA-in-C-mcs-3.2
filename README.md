## Interactive Demos

### Course Visualizer
**URL:** https://ics-2305-data-structures-algorithms-in-c-963520931479.us-west1.run.app/  
Visual demos of the data structures and algorithms covered in this course.

### VisuAlgo
**URL:** https://visualgo.net/en  
Step-by-step animated visualizations of sorting, searching, graphs, and more. Great for watching algorithms execute in real time.  
`sorting` `graphs` `BST` `heap`

### Data Structure Visualizations — USFCA
**URL:** https://www.cs.usfca.edu/~galles/visualization/Algorithms.html  
Classic interactive visualizer from University of San Francisco. Covers AVL trees, B-trees, hash tables, and graph algorithms.  
`AVL trees` `B-trees` `hash tables` `graphs`
